# Tamil Mind Mate - Demo for Clinician Testing

## Deploy to Streamlit Cloud (5 minutes)

1. Go to: **https://share.streamlit.io/**
2. Sign in with GitHub
3. Click "New app"
4. Fill in:
   - Repository: `sahanbull/welcome_mental_health_app`
   - Branch: `main`
   - Main file path: `streamlit-demo/app.py`
5. Click "Deploy!"

Your app will be live in 2-3 minutes with a public URL.

## Features

- ✅ AI conversation (demo responses)
- ✅ 3 grounding exercises with audio
- ✅ UK 3-step safety plan
- ✅ English and Tamil support
- ✅ Distress tracking (SUDS 0-10)
